# C# Tetris

Classic tetris game written in C#, WinForms based.

![Project Screenshot](Screenshot.png)

```bash
git clone https://github.com/xelfiera-shn/csharp-tetris.git
```

## Controls

- **A** - Move left
- **D** - Move right
- **W** - Next rotation
- **Z** - Previous rotation
- **F** - Hold
- **S** - Soft drop
- **Space** - Hard drop
